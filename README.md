Sitio web oficial de **StickerSwap**, la app que conecta coleccionistas para intercambiar figuritas del álbum Panini 2026.

```

│   └── logo.png        # ⚠️ COLOCA TU LOGO AQUÍ

## 💻 Cómo Ver el Sitio Web

### Opción 1: Doble Clic (Más Simple)
1. Navega hasta la carpeta `StickerSwap`
2. Haz doble clic en `index.html`
3. Se abrirá en tu navegador predeterminado

### Opción 2: Desde PowerShell
```powershell
cd "C:\Users\vseno\Desktop\StickerSwap"
Start-Process index.html
```

### Opción 3: Con Servidor HTTP (Recomendado)

**Usando Python:**
```powershell
cd "C:\Users\vseno\Desktop\StickerSwap"
python -m http.server 8000
```
Luego abre en tu navegador: http://localhost:8000

**Usando Node.js (si tienes instalado):**
```powershell
cd "C:\Users\vseno\Desktop\StickerSwap"
npx http-server -p 8000
```

### Opción 4: VS Code + Live Server
1. Abre la carpeta `StickerSwap` en VS Code
2. Instala la extensión "Live Server" (si no la tienes)
3. Clic derecho en `index.html` → "Open with Live Server"
4. Se abrirá automáticamente en tu navegador con recarga automática
